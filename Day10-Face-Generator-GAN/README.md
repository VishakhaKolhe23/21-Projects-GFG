# Face Generator using GAN

## Project Description
This project uses a Generative Adversarial Network (GAN) to generate human face images.
A pretrained generator model is used to generate realistic faces from random noise.

## Key Concepts
- GAN (Generator & Discriminator)
- Latent Space Sampling
- Image Normalization
- Model Inference
