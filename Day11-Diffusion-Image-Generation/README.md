## Project Description
This project demonstrates image generation using state-of-the-art diffusion models from Hugging Face.
A pretrained Stable Diffusion model is used to generate images from text prompts.

## Model Used
- Stable Diffusion v1.5 (`runwayml/stable-diffusion-v1-5`)

## Tools & Libraries
- Python
- Hugging Face Diffusers
- Transformers
- PyTorch
- Matplotlib

## Implementation Steps
1. Authenticate with Hugging Face Hub
2. Load Stable Diffusion pipeline
3. Generate images using text prompts
4. Visualize results using matplotlib

## Sample Prompt
"A beautiful mountain landscape at sunrise, ultra realistic"

## Output
The model generates high-quality images based on textual descriptions.

## Learning Outcomes
- Understanding diffusion-based generative models
- Using Hugging Face pipelines for image generation
- Handling model authentication and inference
